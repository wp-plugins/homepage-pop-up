Just add a simple jquery pop-up on the home page. Add a title and important information on your home page without getting blocked by AdBlock.

== Installation ==

1. Upload 'mlab_popup' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "Homepage Pop-up" and create your first pop-up for your homepage!
